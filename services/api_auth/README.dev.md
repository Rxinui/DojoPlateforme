# API Auth

@author Rxinui

For devs only.

## Installation

Based on **NodeJS v16.13.2** .

```shell
npm i
```

## Environments

Create a `.env` file

```shell
API_AUTH_HOST="localhost"
API_AUTH_PORT="80"
API_DB_HOST="0.0.0.0"
API_DB_USER="sifu"
API_DB_PASSWORD="sifu"
API_DB_DATABASE="dojo"
API_DB_CONNECTION_LIMIT=10
```
