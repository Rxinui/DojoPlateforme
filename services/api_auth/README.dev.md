# API Auth

@author Rxinui

For devs only.

## Installation

Based on **NodeJS v16.13.2** .

```shell
npm i
```

## Environments

Create a `.env` file

```shell
API_HOST_URL="http://localhost:3000"
API_DB_HOST="0.0.0.0"
API_DB_USER="root"
API_DB_PASSWORD="root"
API_DB_DATABASE="dojo"
API_DB_CONNECTION_LIMIT=10
```
